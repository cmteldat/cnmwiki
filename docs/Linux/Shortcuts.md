# Shortcuts
```bash
# Move between workspaces (Right/Left)
Ctrl + Alt + Right/Left

# Move window between workspaces (Right/Left)
Ctrl + Super + Right/Left

# Move window between monitors (Right/Left)
Alt + Shift + Right/Left

# List of applications
Alt + Space

# Activities menu
Super (Windows Key)

# View notifiacions
Super + V
Super + M

# Open Terminal
Ctrl + Alt + T

# Lock PC
Super + L
```
